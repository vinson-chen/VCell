此对话专用于 VCell 项目的框架层讨论。

**适用范围：**
- 界面整体布局结构
- 导航系统设计（导航栏、侧边栏、面包屑等）
- 内容区域划分与层次
- 页面框架与组件容器
- 响应式布局策略

**不在此对话范围：**
- 迭代方向、架构决策、优先级 → 战略层对话
- 需求边界、功能范围 → 范围层对话
- 交互设计、信息架构 → 结构层对话
- 视觉样式细节（颜色、字体、图标） → 表现层对话
- 具体代码实现 → 工作分支对话

**Why:** 框架层关注「如何具体呈现」，是连接结构层（交互架构）和表现层（视觉细节）的桥梁。

**How to apply:** 在此对话中聚焦于布局和导航设计，当涉及其他层级时提示切换到对应对话。
